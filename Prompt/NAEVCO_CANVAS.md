# NAEVCO — Business Model Canvas

> **Misión:** Hacer verificable, independiente y accionable la descarbonización de las cadenas de suministro global mediante datos satelitales.

---

## 1. Segmentos de Clientes

**Segmento A — Shippers Corporativos**
Empresas multinacionales con metas Scope 3 públicas y presión regulatoria para auditarlas. Buscan protección legal y credibilidad ESG.
- Tamaño del mercado: ~4,000 empresas globales con compromisos SBTi activos
- Ejemplos: Amazon, IKEA, Unilever, Nestlé

**Segmento B — Autoridades Portuarias**
Operadores de puertos comerciales y ministerios de transporte que necesitan métricas de impacto verificables para atraer inversión y cumplir regulaciones.
- Tamaño del mercado: ~900 puertos con tráfico significativo (WPI)
- Ejemplos: Port of Rotterdam, Santos, Cartagena, Singapore

**Segmento C — Instituciones Financieras ESG**
Fondos de inversión, bancos de desarrollo, aseguradoras que necesitan verificar el impacto real de sus portfolios de activos logísticos "verdes".
- Tamaño del mercado: $50B+ en flujos ESG hacia logística/infraestructura anual
- Ejemplos: BlackRock, BID Invest, IFC, Allianz

---

## 2. Propuesta de Valor

### Propuesta central
**Verificación satelital independiente de emisiones logísticas.** Lo que NAEVCO mide no puede ser alterado por ningún actor de la cadena — los datos vienen directamente de satélites de la ESA y NOAA.

### Por segmento

**Para Shippers:**
- Evidencia auditable de Scope 3 para reguladores y accionistas
- Protección contra acusaciones de greenwashing
- Comparador de rutas y navieras por huella real para procurement

**Para Puertos:**
- Score de desempeño ambiental vs. competidores (benchmarking objetivo)
- Métricas de impacto de inversiones en infraestructura verde
- Reportes listos para financiadores multilaterales (BID, Banco Mundial)

**Para Financieras:**
- Due diligence satelital pre-inversión en activos logísticos
- Monitoreo continuo de riesgo ambiental en portfolio
- Datos para ratings ESG verificables y defendibles ante reguladores

### Diferenciación clave
1. **Independencia total**: nadie en la cadena puede manipular los datos satelitales
2. **Cobertura multimodal**: único sistema que conecta marítimo + puertos + terrestre en un solo score
3. **Resolución temporal**: actualización mensual vs. reportes anuales de la industria
4. **Accionabilidad**: no solo mide, identifica el bottleneck y sugiere dónde invertir

## 3. Canales

### Canales de venta

**Directo (prioritario para los primeros 18 meses):**
- Equipo de ventas propio enfocado en las 3 verticales
- Demos personalizadas con datos reales del cliente (mostrar su propia ruta, su propio puerto)
- Conferencias del sector: Nor-Shipping, Europort, CDP Summit, GreenBiz

**Partnerships (para escalar):**
- Integraciones con plataformas ESG existentes (Watershed, Persefoni, Sphera)
  → Ellos ya tienen al shipper como cliente, NAEVCO entra como módulo de verificación
- Organismos internacionales como canales (UNCTAD, IMO, CAF)
  → Pueden empujar adopción en países en desarrollo como estándar

**Indirecto:**
- Consultoras de sostenibilidad (Deloitte, PwC, Accenture) que recomienden NAEVCO a sus clientes ESG
- Auditoras que certifiquen reportes de emisiones y necesiten la fuente de datos

### Canal de producto (autoservicio)
- Dashboard web con free tier para 1 corredor (lead generation)
- API con documentación pública para equipos técnicos de clientes

## 4. Relación con Clientes

**Shippers:**
- Onboarding asistido (configurar sus corredores, calibrar con sus datos propios)
- Customer Success mensual (revisar el reporte juntos, interpretar alertas)
- Integración con sus plataformas ESG existentes (reduce fricción de adopción)

**Puertos:**
- Relación consultiva (no solo vender datos, ayudar a interpretarlos para pitch de financiadores)
- Workshops de benchmark semestral comparando su puerto vs. competidores
- Posibilidad de co-publicar estudios de caso (marketing para ellos y para NAEVCO)

**Financieras:**
- Modelo de datos-as-a-service con SLA de calidad y disponibilidad
- Soporte analítico para interpretar datos en contexto de decisiones de inversión
- Relación de bajo contacto humano pero alta confiabilidad de datos

## 5. Fuentes de Ingreso

### Ingresos recurrentes (80% objetivo)

**SaaS por corredor monitoreado:**
- Starter: $1,500/mes — 2 corredores, reportes PDF
- Professional: $4,500/mes — 8 corredores, API, alertas
- Enterprise: $12,000+/mes — ilimitado, metodología co-auditada

**Suscripción institucional (puertos):**
- Monitor: $3,000/mes — 1 puerto
- Compete: $7,500/mes — 1 puerto + benchmarking vs. 10
- Institutional: $60,000–$150,000/año — marco multi-puerto

**Data license (financieras):**
- API access: $8,000/mes — datos de todos los corredores
- Enterprise: $80,000–$250,000/año — dataset completo

- $5,000–$15,000/año por corredor o puerto certificado

## 6. Recursos Clave

### Datos (el activo más importante)
- Acceso a Google Earth Engine (gratuito para proyectos de investigación/no-profit inicialmente)
- Sentinel-5P y VIIRS: datos públicos, costo = procesamiento computacional
- World Port Index, OECD, IMO DCS: públicos y gratuitos
- AIS data: costo ~$500–$2,000/mes para feed comercial en escala

### Tecnología
- Pipeline de procesamiento geoespacial (GEE + Python)
- Dashboard web (React + Deck.gl/Mapbox)
- API de datos (FastAPI)
- Base de datos de series temporales (PostgreSQL + TimescaleDB o InfluxDB)
- Infraestructura cloud (Google Cloud Platform, por integración natural con GEE)

### Propiedad intelectual
- **Metodología de scoring**: la fórmula que convierte NO₂ satelital en score de corredor — esto es el activo defensible central
- **Modelos de calibración**: correlaciones entre señal satelital y emisiones reales construidas con datos históricos
- **Dataset histórico**: 5+ años de datos procesados de todos los corredores monitoreados

### Capital humano
- Científico/a geoespacial senior (calibración de modelos, validación de datos)
- Desarrollador full-stack (plataforma web + API)
- Data engineer (pipelines de procesamiento)
- Sales + relaciones institucionales (1-2 personas)
- CEO con conocimiento de logística/marítimo o sostenibilidad (credibilidad ante clientes)

---

## 7. Actividades Clave

### Producción de datos (núcleo operacional)
- Procesamiento mensual automatizado de Sentinel-5P y VIIRS sobre corredores y puertos
- Control de calidad de datos (filtrado de nubes, validación de anomalías)
- Actualización y recalibración de modelos de scoring
- Incorporación de nuevos corredores y puertos según demanda de clientes

### Producto
- Desarrollo y mantenimiento del dashboard web
- API disponible 24/7 con 99.5%+ uptime
- Integración con plataformas ESG de terceros (Watershed, etc.)
- Nuevas features según feedback de clientes

### Validación y credibilidad
- Publicación de metodología (white paper técnico disponible públicamente)
- Auditoría de metodología por tercero independiente (año 1)
- Acreditación ante organismos internacionales (IMO, UNCTAD)
- Estudios de caso publicados con clientes

### Ventas y relaciones
- Pipeline de prospectos en los 3 segmentos
- Demos personalizadas con datos reales de cada prospecto
- Participación en conferencias sectoriales clave
- Alianzas con plataformas ESG y consultoras

## 8. Socios Clave

### Socios de datos
- **ESA / Copernicus Programme**: fuente del Sentinel-5P (relación a construir para posicionamiento, datos son públicos)
- **NOAA**: fuente de VIIRS
- **Google Earth Engine**: infraestructura de procesamiento — explorar Google for Startups para créditos
- **Proveedor AIS**: MarineTraffic, exactEarth, o Spire Maritime (para datos en tiempo real)

### Socios de distribución
- **Plataformas ESG** (Watershed, Persefoni, Salesforce Net Zero Cloud): integrarse como módulo de verificación marítima dentro de sus plataformas — ellos ya tienen los shippers como clientes
- **Consultoras de sostenibilidad** (Deloitte, Accenture, PwC): recomendar NAEVCO en sus proyectos de estrategia ESG
- **Auditoras de emisiones**: ellas necesitan fuentes de datos verificables para certificar reportes

### Socios institucionales
- **IMO**: validación de metodología, posible integración con DCS data
- **CAF / BID**: para llegar a puertos latinoamericanos como parte de programas de modernización
- **World Ports Organization (WPO)**: acceso a red global de puertos miembros
- **Asociaciones de navieras** (BIMCO, ICS): para construir credibilidad en el sector

### Socios tecnológicos
- **Google Cloud / Google for Startups**: créditos de infraestructura para GEE y compute
- **Esri / ArcGIS**: herramientas de análisis geoespacial avanzado

## 9. Estructura de Costos

### Costos fijos (mensuales)

| Concepto | Costo mensual estimado |
|---|---|
| Google Earth Engine / GCP compute | $500–$1,500 |
| AIS data feed (MarineTraffic / Spire) | $500–$2,000 |
| Infraestructura cloud (DB, API hosting) | $300–$800 |
| Herramientas SaaS (Figma, Notion, CRM) | $200 |
| **Total infraestructura** | **$1,500–$4,500/mes** |

### Costos de equipo (headcount mínimo viable)

| Rol | Perfil | Costo mensual (LatAm) |
|---|---|---|
| CEO/Founder | Estrategia, ventas | — (equity) |
| Geoespacial/Data Scientist | Modelos y pipelines | $2,500–$4,000 |
| Full-stack Developer | Producto web + API | $2,500–$4,000 |
| Sales / BD | Ventas y relaciones | $2,000–$3,500 |
| **Total equipo (mínimo)** | | **$7,000–$11,500/mes** |

### Costos variables
- Procesamiento adicional GEE por cliente nuevo: ~$50–200/cliente/mes
- Costo de adquisición por cliente (eventos, demos, viajes): ~$3,000–$8,000 por deal enterprise
- Auditoría externa de metodología (1 vez, año 1): $15,000–$30,000

### Burn rate total estimado
- **Pre-revenue (primeros 6 meses):** ~$10,000–$16,000/mes
- **Con primeros clientes (6-18 meses):** ~$15,000–$22,000/mes

### Punto de equilibrio
Con el modelo conservador de Año 1 ($782,000 ARR = ~$65,000/mes), el punto de equilibrio operacional se alcanza alrededor del mes 10-12 con 3-4 clientes pagantes activos.
