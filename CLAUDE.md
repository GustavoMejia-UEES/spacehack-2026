## Tech Stack

- **React 19 + TypeScript** with Vite 7
- **TanStack Start** — full-stack framework (SSR-capable); **TanStack Router** — file-based routing
- **Three.js + @react-three/fiber + @react-three/drei** — 3D character rendering
- **Tailwind CSS 4** via Vite plugin
- **Zustand** — installed but currently unused; reserved for future global state

## Architecture

### Routing

File-based routing via TanStack Router. Drop a file in `src/routes/` and the router picks it up automatically. The route tree is auto-generated at `src/routeTree.gen.ts` — do not edit this file manually.

- Root layout: `src/routes/__root.tsx` — global HTML shell, Header, Footer, theme init script
- Router config: `src/router.tsx`

### Feature Modules

Code is organized into feature folders under `src/features/`. Each feature owns its components, hooks, types, and pages.

The only active feature is `src/features/character/` — a standalone 3D virtual assistant module designed to be copy-paste portable to other React projects.

### 3D Character Module (`src/features/character/`)

**Core component stack:**

```
FloatingCharacter3D       ← main entry point (wraps Canvas + ChatBubble)
├── Canvas (R3F)
│   ├── GenericModel      ← loads any .glb, auto-detects animations via useGLTF/useAnimations
│   ├── Float             ← floating idle motion
│   ├── ContactShadows    ← shadow
│   └── Environment       ← auto-lighting
└── ChatBubble            ← DOM overlay (positioned absolutely over Canvas)
```

**Usage:**
```jsx
<FloatingCharacter3D
  modelPath="/models/MyModel.glb"
  position="bottom-right"   // bottom-right | bottom-left | top-right | top-left
  size="md"                 // sm | md | lg
  isAnimating={isAnimating}
  message={activeMessage}
/>
```

**Animation:** `GenericModel` auto-detects all animations in the `.glb` file (logs them to console as `📦 Animaciones detectadas: [...]`). Transitions use `THREE.AnimationMixer.crossFadeTo(0.5s)`. The `isAnimating` prop toggles between idle/active states.

**Message state hook:**
```ts
const { activeMessage, isAnimating, dismissMessage } = useSatelliteMessages(true)
```
`useSatelliteMessages` rotates mock messages every 12s and auto-dismisses after 6s. Supports external messages (e.g., from N8N or Socket.io).

**ChatBubble** has three color variants: `alert` (red), `warning` (orange), `info` (blue). Mobile layout: centered top, 90% width. Desktop: positioned near the character's head using a 3D→2D coordinate projection via `HeadPositionTracker`.

### Path Aliases

Both `#/*` and `@/*` resolve to `./src/*` (configured in `vite.config.ts` and `tsconfig.json`).

### Code Style

Prettier config: no semicolons, single quotes, trailing commas. TypeScript strict mode, ES2022 target.

### 3D Models

Place `.glb` files in `public/models/`. Export from Blender with animations baked in. See `PERSONAJE_3D_SETUP.md` and `src/features/character/HOW_TO_CHANGE_MODEL.md` for details.
